Part 1: Variables & Conditionals

JavaScript variables are declared using let, const, or var.

// Variable declarations
let score = 85;
const passingGrade = 60;
var message = "";

// Conditional statements
if (score >= 90) {
  message = "Excellent!";
} else if (score >= passingGrade) {
  message = "You passed!";
} else {
  message = "Please try again.";
}

Part 2: Custom Functions

Functions are reusable blocks of code that perform a specific task.

// Function to calculate area of a rectangle
function calculateArea(width, height) {
  return width * height;
}

// Function to check if a number is even
const isEven = num => num % 2 === 0;

Part 3: Loops

Loops execute a block of code multiple times.

// For loop example
for (let i = 0; i < 5; i++) {
  console.log("Iteration: " + i);
}

// While loop example
let count = 0;
while (count < 3) {
  console.log("Count: " + count);
  count++;
}

Part 4: DOM Interactions

The Document Object Model (DOM) allows JavaScript to interact with HTML.

// Changing text content
document.getElementById('text').textContent = 'New text!';

// Changing styles
document.getElementById('box').style.backgroundColor = 'blue';

// Handling button clicks
button.addEventListener('click', function() {
  alert('Button clicked!');
});
